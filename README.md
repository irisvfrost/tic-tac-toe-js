# Tic-Tac-Toe (2 Player)

A simple 2-player Tic-Tac-Toe game that runs directly in the browser. Built with vanilla HTML, CSS, and JavaScript.

## Live Demo
- https://irisvfrost.github.io/tic-tac-toe-js/

## Features
- Two-player gameplay (X vs O)
- Win detection + highlighted winning line
- Draw detection
- Reset button

## Tech Stack
- HTML
- CSS
- JavaScript (DOM + event handling)

## Run Locally
1. Download the repo
2. Open `index.html` in your browser
